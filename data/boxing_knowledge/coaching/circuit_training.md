# Boxing Circuit Training Guide for Coaches

This guide is for coaches running boxing circuit training classes with the BoxBunny system. Circuit training is a time-efficient format that works well for group classes, providing both technical practice and cardiovascular conditioning.

## Class Structure Overview

A typical boxing circuit class runs 45-60 minutes and follows this structure:

1. **Warm-up** (8-10 min)
2. **Technical Instruction** (5 min)
3. **Circuit Rounds** (25-35 min)
4. **Core/Finisher** (5-8 min)
5. **Cool-down/Stretch** (5 min)

## Warm-up Design

The warm-up should progressively increase heart rate and mobilize boxing-specific joints.

### Sample Warm-up (10 min)
| Time | Activity |
|------|----------|
| 0:00-2:00 | Light jogging / skipping in place |
| 2:00-4:00 | Dynamic arm circles, shoulder rolls, hip circles |
| 4:00-6:00 | Shadow boxing -- jabs only, light and loose |
| 6:00-8:00 | Shadow boxing -- add crosses and hooks, increase pace |
| 8:00-10:00 | Footwork drills: step-drag forward/back/lateral |

### Key Points
- Never start a class by hitting the bag cold.
- Include wrist rotations and finger spreads after wrapping hands.
- Use the warm-up to check that everyone has proper hand wraps and gloves.

## Station Design Principles

### Balance of Modalities
Each circuit should include:
- **Punching stations**: Bag work, pad work, or BoxBunny robot drills
- **Conditioning stations**: Bodyweight exercises, jump rope, agility
- **Active recovery stations**: Light shadow boxing, footwork, stretching

### Work-to-Rest Ratios
| Class Level | Work | Rest | Rounds |
|-------------|------|------|--------|
| Beginner | 45 sec | 30 sec | 3 |
| Intermediate | 60 sec | 20 sec | 4 |
| Advanced | 90 sec | 15 sec | 5 |

### Station Rotation
- Use a whistle or bell to signal rotation (the BoxBunny system timer can synchronize with class timing).
- Allow 10-15 seconds of transition time within the rest period.
- Participants rotate clockwise through stations.

## Sample Beginner Circuit (6 Stations, 3 Rounds)

| Station | Exercise | Focus |
|---------|----------|-------|
| 1 | Heavy bag -- jab-cross only | Technique: straight punches |
| 2 | Jump rope | Conditioning: calf endurance |
| 3 | Shadow boxing with footwork | Technique: movement + guard |
| 4 | Push-ups (knees OK) | Conditioning: pushing strength |
| 5 | Heavy bag -- 1-2-3 combo | Technique: adding hooks |
| 6 | Plank hold | Conditioning: core stability |

Work: 45 seconds per station. Rest: 30 seconds. Total circuit time: 3 rounds x (6 stations x 75 sec) = ~22 min.

## Sample Intermediate Circuit (8 Stations, 4 Rounds)

| Station | Exercise | Focus |
|---------|----------|-------|
| 1 | Heavy bag -- power combinations | Technique: power + hip rotation |
| 2 | Burpees | Conditioning: full body |
| 3 | Double-end bag / BoxBunny reaction drill | Technique: timing + reflexes |
| 4 | Squat jumps | Conditioning: explosive legs |
| 5 | Heavy bag -- body work | Technique: level changes |
| 6 | Mountain climbers | Conditioning: cardio + core |
| 7 | Shadow boxing with slips | Technique: defense integration |
| 8 | Russian twists with medicine ball | Conditioning: rotational core |

Work: 60 seconds per station. Rest: 20 seconds. Total circuit time: 4 rounds x (8 stations x 80 sec) = ~43 min.

## Sample Advanced Circuit (8 Stations, 5 Rounds)

| Station | Exercise | Focus |
|---------|----------|-------|
| 1 | Heavy bag -- freestyle combinations | Technique: flow and improvisation |
| 2 | Box jumps or tuck jumps | Conditioning: explosive power |
| 3 | BoxBunny defence drill (intermediate+) | Technique: blocking and slipping |
| 4 | Battle ropes | Conditioning: shoulder endurance |
| 5 | Heavy bag -- volume round (max punches) | Conditioning: punch stamina |
| 6 | Plyometric push-ups | Conditioning: explosive pushing |
| 7 | BoxBunny shadow sparring (intermediate combos) | Technique: combination accuracy |
| 8 | Ab wheel rollouts | Conditioning: anti-extension core |

Work: 90 seconds per station. Rest: 15 seconds. Total circuit time: 5 rounds x (8 stations x 105 sec) = ~70 min (including transitions).

## Managing Different Skill Levels

In a mixed-level class, provide variations at each station:

- **Bag work**: Beginners throw 1-2 only. Intermediates throw 1-2-3. Advanced throw freestyle.
- **Conditioning**: Beginners do modified versions (push-ups on knees, step-ups instead of box jumps).
- **BoxBunny**: Set difficulty level per participant using the coach dashboard.

## BoxBunny Integration for Circuit Classes

### Coach Dashboard Features
- Set round timers that synchronize across all stations
- Monitor each participant's punch count and accuracy in real time
- Assign different drill sequences to different skill levels
- Review session summaries for the entire class after completion

### Recommended BoxBunny Stations
1. **Reaction Time Station**: Participants compete for best reaction time. The leaderboard drives motivation.
2. **Combo Accuracy Station**: A specific combination is displayed. The participant must execute it correctly within the time window.
3. **Defence Station**: The robot throws punches and the participant must block or slip. Score is based on defense rate.

## Safety Considerations

1. **Hydration**: Place water stations near the circuit. Encourage drinking during rest periods.
2. **Form Over Speed**: Remind participants that proper technique takes priority over volume.
3. **Fatigue Watch**: Monitor participants for form breakdown in later rounds. Call out corrections.
4. **Hand Protection**: Check that wraps are secure at the start. Re-wrap between rounds if needed.
5. **Space**: Ensure adequate distance between stations to prevent collisions.
6. **Scalability**: Always provide easier alternatives for each exercise. Not everyone needs to do burpees.

## Finisher Ideas

End the class with a short, intense finisher to create a sense of accomplishment:

- **Punch-out**: 30 seconds of maximum-speed straight punches on the bag.
- **Plank Challenge**: Hold plank as long as possible as a class. Last person standing wins.
- **Tabata Shadow Boxing**: 20 seconds max effort, 10 seconds rest, 8 rounds (4 minutes total).
- **Team Relay**: Two teams race to accumulate 500 punches on the bag.
